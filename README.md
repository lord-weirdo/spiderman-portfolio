# Pratham Desai - 3D Portfolio

Welcome to the source code for my personal portfolio! This is a highly interactive, 3D animated website built to showcase my projects, experience, and development skills.

## ✨ Features
- **3D Interactive Model:** Features a high-quality Spider-Man 3D model integrated using React Three Fiber.
- **Custom Scroll Animations:** Uses GSAP (GreenSock) for smooth vertical and horizontal scroll-linked animations.
- **Dynamic Project Carousel:** A custom-built, responsive horizontal scroll section to showcase my latest work.
- **Fully Responsive:** Beautifully tailored UI for desktop, tablet, and mobile devices.

## 🚀 Technologies Used
- **Framework:** React.js + Vite
- **3D Rendering:** Three.js, `@react-three/fiber`, `@react-three/drei`
- **Animations:** GSAP (ScrollTrigger, SplitText)
- **Styling:** Vanilla CSS with modern custom properties

## 🛠️ Running Locally
To run this project on your local machine:

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
